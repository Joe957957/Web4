# 台鐵票價觀察站

這是一個以「台灣鐵路票價」為主題的物價觀察網站，符合以下需求：

- 前端網頁
- Express.js Web API
- SQLite 資料庫
- 新增資料：日期、商品名稱、商品價格、資料來源網站
- 簡易查詢：關鍵字、商品名稱、日期區間
- 表格化呈現票價變化

## 專案結構

- `server.js`：Express 伺服器與 API
- `db.js`：SQLite 連線與資料表初始化
- `public/`：前端畫面與互動程式
- `openspec/`：實作過程與規格紀錄
- `docs/`：可貼到 HackMD / Notion 的教學文件稿

## 主題說明

我選擇台鐵票價作為通貨膨脹觀察主題，原因是鐵路票價和民生交通支出有直接關聯，也容易從官方公告、新聞與票價查詢頁面追蹤變化。這個主題適合做成資料表與查詢系統，能讓使用者快速比較不同日期的票價記錄。

## 執行方式

如果你之後要在本機啟動，先安裝相依套件，再執行：

```bash
npm install
npm start
```

然後開啟：

```text
http://localhost:3000
```

## 補充

- SQLite 資料庫會在第一次啟動時自動建立於 `data/rail_fares.sqlite3`
- 目前前端已經包含新增、查詢與表格列表
- 若要交作業，建議先補上真實票價資料與截圖，再貼到 HackMD 文件
