# 截圖補充說明

本目錄已準備好三個截圖的佔位符，文件中指向 `images/` 子目錄：

## 需要補充的截圖

1. **01-首頁畫面.png** - 首頁整體（顯示表單、查詢區、表格與統計摘要）
2. **02-新增資料成功.png** - 新增一筆資料後的成功提示畫面
3. **03-查詢結果.png** - 使用查詢功能（搜尋特定商品）後的表格結果

## 如何補上截圖

### 方案 1：手動截圖（推薦）

1. 啟動網站：`npm start`
2. 開瀏覽器進入 `http://localhost:3000`
3. 分別截三個畫面：
   - 整個首頁
   - 在表單填入資料後按「儲存紀錄」
   - 在查詢條件填入文字後按「重新查詢」
4. 將三張截圖存到本目錄 `images/` 下，並命名為上述三個檔名

### 方案 2：上傳到 HackMD / Notion

如果你使用 HackMD 或 Notion：
1. 直接在文件中拖放或上傳三張截圖
2. HackMD 會自動生成 URL，無需存在本地

### 方案 3：上傳到 GitHub 後再引用

1. 先把整個專案推到 GitHub（已完成 ✓）
2. 在 GitHub 上上傳三張截圖到 `docs/images/` 目錄
3. 文件中的相對連結會自動指向 GitHub 上的圖片

## 檔案路徑範例

```
web4/
├── docs/
│   ├── images/
│   │   ├── 01-首頁畫面.png
│   │   ├── 02-新增資料成功.png
│   │   └── 03-查詢結果.png
│   ├── hackmd-template.md
│   └── SCREENSHOTS_README.md (本檔案)
```

## 備註

- 若使用 HackMD，可直接將截圖拖到文件中，無需放在 images 目錄
- 若使用 Notion，亦可直接上傳圖片
- 若最後要分享 GitHub 連結給老師，建議把圖片一起推上去
